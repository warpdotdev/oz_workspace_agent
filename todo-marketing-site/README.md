# SimpleDo Marketing Website

Marketing website for the SimpleDo to-do application.

## Features

- Hero section with app preview mockup
- Features section highlighting 3 key benefits
- Testimonials/social proof section
- About section with product philosophy
- CTA section
- Responsive footer with navigation

## Tech Stack

- React 19
- Vite 7
- CSS with custom properties for theming

## Design System

Colors:
- Primary: #4A90D9 (calm blue)
- Success: #34C759 (soft green)
- Background: #FAFAFA
- Card: #FFFFFF
- Text Primary: #1A1A1A
- Text Secondary: #6B6B6B

Typography: Inter font family

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Messaging

- Tagline: "Do more. Manage less."
- Hero: "Tired of productivity tools that need their own tutorials?"
- Value prop: Simplicity, speed, privacy
